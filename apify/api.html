1
Returns a json list of cities
REQUEST
GET /cities.json?my_data=1
application/json

RESPONSE
application/json; charset=utf-8
[
  {
    "city": {
      "country": "Bangladesh",
      "created_at": "2011-08-23T05:30:03Z",
      "id": 1,
      "name": "Dhaka",
      "updated_at": "2011-08-23T05:30:03Z"
    }
  }
]

2
Creates a city with json data
REQUEST
POST /cities.json
application/json
{
  "city": {
    "name": "Toronto",
    "country": "Canada"
  }
}

RESPONSE
application/json; charset=utf-8
{
  "city": {
    "country": "Canada",
    "created_at": "2011-08-23T05:30:03Z",
    "id": 2,
    "name": "Toronto",
    "updated_at": "2011-08-23T05:30:03Z"
  }
}
